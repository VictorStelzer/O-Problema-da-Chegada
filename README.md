# O Problema da Chegada

O Problema da Chegada é um jogo criado por mim como parte de um trabalho da faculdade. Ele transforma temas de computação em uma aventura espacial leve e divertida. No jogo, os jogadores embarcam em uma jornada através do cosmos, viajando por um tabuleiro curvado que lembra uma trilha galáctica. A estética mistura nebulosas, luzes estelares e um clima futurista, criando a sensação de estar realmente navegando pelo espaço. É uma experiência competitiva, descontraída e cheia de personalidade, pensada para tornar o estudo mais visual e mais divertido.
